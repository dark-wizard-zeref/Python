# File & Folder Manager (Python)

A simple command-line based file and folder management system written in Python.  
This project allows users to create, read, update, and delete files and folders using a menu-driven interface.

---

## Features

### Folder Operations
- Create a folder  
- List all files and folders (recursively)  
- Rename a folder  
- Delete a folder (even if it is not empty)  

### File Operations
- Create a file  
- Read a file  
- Rename a file  
- Append content to a file  
- Overwrite file content  
- Delete a file  

---

## Requirements
- Python 3.x or higher  
- No external libraries required  

---

## How to Run

1. Open a terminal in the project directory  
2. Run the program:

```bash
python main.py
```
